## Template

```markdown
# ADR-XXX: [Short title]

## Status
[Proposed | Accepted | Deprecated | Superseded by ADR-YYY]

## Context
[Describe the situation that motivated the decision]

## Decision
[State the decision clearly]

## Consequences
[Describe the resulting context after applying the decision]
```
